# React-redux 实现的计数器
### 如何执行
####  1、将项目克隆到本地，cd到Node-Express-MongoDB
```javascript
git clone git@github.com:jonechen1986/Node-Express-MongoDB.git
cd Node-Express-MongoDB
```
#### 2、安装依赖
```javascript
npm install
```
#### 3、执行
```javascript
node index.js
```
或
```javascript
npm start
```
#### 4、打开浏览器浏览 http://localhost:3000/
